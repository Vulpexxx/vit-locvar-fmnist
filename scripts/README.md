# 脚本目录说明

这里放可复用的实验脚本，不放最终图片结果。

## 命名建议

- `plot_training_curves.py`：训练/验证曲线
- `plot_patch_feature_comparison.py`：patch 特征对比
- `plot_attention_maps.py`：注意力可视化
- `plot_model_comparison.py`：模型对比图
- `plot_ablation_results.py`：消融实验结果图

## 结果保存建议

- 图片优先保存到 `visualizations/`
- 如果是某个实验专属图，保存到对应实验目录下的 `plots/`
